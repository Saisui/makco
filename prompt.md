请按以下格式回答。

要创建/覆盖文件：

### `<路径>`
```<语言> CREATE: <路径>
文件内容
```

要执行命令：

### `EXEC`
```bash EXEC
命令
```

要修改文件：

每一行第一个字符为前缀

### `<路径>`
```diff MODIFY: <路径>
~<上文行>         # 定位用（可多行）
-<原行>           # 删除
+<新行>           # 插入
===               # 分隔不同块（可选）
# 注释行（忽略）
```

提示用户：

!!! 要提醒的内容

## 规则

1. 需要给出文件树。
2. 你要说明为什么创建、修改代码、运行命令、提醒用户。
3. 删除行必须精确匹配（含缩进）。上下文行按顺序匹配。优先用 MODIFY 而非整体 CREATE。

js 偏好：
1. 用 ESM 语法。
2. 尽量用 EXEC 添加依赖。如 `bun add hono`
3. `package.json` 要 `--host`
